AutoIt Test Framework

AutoIt Test Framework is a simple, lightweight unit test framework for AutoIt. It provides colored console output, pass/fail tracking, and cumulative test result accumulation with zero dependencies beyond AutoIt itself.

Designed to be used alongside any AutoIt library or script, AutoIt Test Framework makes it easy to write, run, and read test suites directly inside SciTE's console output pane.

Key Features

Colored console outputPass/fail results are color-coded in SciTE's output pane for at-a-glance readability.
Actual vs. expected displayOn failure, the actual and expected values are printed alongside the description.
Cumulative pass/fail tracking_TestFmkRun() accumulates results across multiple test functions, so a single $bAllPassed variable reflects the entire suite.
Summary block_TestFmkSummary() prints a final total/passed/failed count in the AutoIt convention used by Au3Check output.
Testable wrappers and stubsTestable.au3 and Stubs.au3 provide a pattern to test code that uses GUI controls, dialogs, file system, registry, and other side-effecting AutoIt built-ins — without those operations actually executing during tests.
Zero dependenciesThe core framework requires no #include beyond TestFramework.au3 itself. Testable.au3 and Stubs.au3 are additional opt-in libraries for testing GUI and side-effecting code.

Quick Example

#include <TestFramework.au3>

Func _TestMyFunc()
    _TestFmkHeader("Test: MyFunc")
    _TestFmkAssert(MyFunc(2, 3) = 5, "MyFunc(2, 3) returns 5", MyFunc(2, 3), "5")
    _TestFmkAssert(MyFunc(0, 0) = 0, "MyFunc(0, 0) returns 0", MyFunc(0, 0), "0")
EndFunc

Func _RunAllTests()
    Local $bAllPassed = True
    $bAllPassed = _TestFmkRun(_TestMyFunc, $bAllPassed)
    _TestFmkSummary()
EndFunc

_RunAllTests()

Console output:

+ --- Test: MyFunc ---
> [PASS] MyFunc(2, 3) returns 5
> [PASS] MyFunc(0, 0) returns 0

+ Total: 2
> Passed: 2
: Failed: 0

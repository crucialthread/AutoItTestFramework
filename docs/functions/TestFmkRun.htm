_TestFmkRun

Runs a test function by reference and accumulates its pass/fail result into a cumulative boolean.

#include <TestFramework.au3>
_TestFmkRun ( $fTest [, $bCumulative = True] )

Parameters

$fTestA function reference to the test function to run. Pass the function name directly without parentheses or quotes (e.g. _TestMyFunc, not "_TestMyFunc" or _TestMyFunc()).
$bCumulative[optional] The accumulated result from all previously run tests. Pass $bAllPassed here to carry the running total forward. Default is True.

Return Value

Success:True if this test produced no new failures AND $bCumulative is True.
Failure:False if this test produced at least one new failure, OR if $bCumulative was already False.

Remarks

_TestFmkRun() snapshots the internal failure counter before calling $fTest(), then compares it afterward. If the count increased, the test is considered failed and False is returned.

Critically, _TestFmkRun() always runs every test function, regardless of whether previous tests have already failed. This is intentional: the full suite should always complete so you get a complete picture of what passes and fails in a single run, rather than stopping at the first failure.

The standard pattern is to assign the return value back to $bAllPassed on each call, chaining the cumulative result through every test in the suite: 

Local $bAllPassed = True
$bAllPassed = _TestFmkRun(_TestA, $bAllPassed)
$bAllPassed = _TestFmkRun(_TestB, $bAllPassed)
$bAllPassed = _TestFmkRun(_TestC, $bAllPassed)

After all tests have run, $bAllPassed is True only if every single test in the suite passed. This makes it usable as a return value from the overall _RunAllTests() function for external callers that need a single pass/fail signal.

Related

_TestFmkHeader, _TestFmkAssert, _TestFmkSummary

Example

#include <TestFramework.au3>

Func _TestAdd()
    _TestFmkHeader("Test: Add()")
    _TestFmkAssert(1 + 1 = 2, "1 + 1 = 2", 1 + 1, "2")
EndFunc

Func _TestSubtract()
    _TestFmkHeader("Test: Subtract()")
    _TestFmkAssert(5 - 3 = 2, "5 - 3 = 2", 5 - 3, "2")
EndFunc

Func _RunAllTests()
    Local $bAllPassed = True
    $bAllPassed = _TestFmkRun(_TestAdd,      $bAllPassed)
    $bAllPassed = _TestFmkRun(_TestSubtract, $bAllPassed)
    _TestFmkSummary()
    Return $bAllPassed
EndFunc

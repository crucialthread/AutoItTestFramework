_TestFmkAssert

Evaluates a condition, records pass or fail, and prints the result to the console.

#include <TestFramework.au3>
_TestFmkAssert ( $bCondition, $sDescription [, $vActual = "" [, $vExpected = ""]] )

Parameters

$bConditionThe condition to evaluate. If True, the assertion passes; if False, it fails.
$sDescriptionA short description of what this assertion is checking. Printed alongside the pass/fail result.
$vActual[optional] The actual value produced by the code under test. Displayed on failure so you can see what the code returned. Strongly recommended.
$vExpected[optional] The expected value. Displayed on failure alongside $vActual. Strongly recommended.

Return Value

None.This function does not return a value. Pass/fail state is tracked in internal global counters.

Remarks

Every call to _TestFmkAssert() increments the internal total counter. A passing assertion increments the passed counter and prints a blue [PASS] line. A failing assertion increments the failed counter and prints a red [FAIL] line.

Always provide $vActual and $vExpected. Without them, a failure only tells you which description failed, not what the actual value was. With them, you can diagnose the failure directly from the console output without re-running the script under a debugger.

Important: if you are asserting on @error, capture it into a local variable immediately after the call being tested, before passing it to _TestFmkAssert(). Any function call (including _TestFmkAssert itself) resets @error to 0.

Correct: Local $iErr = @error (captured immediately), then _TestFmkAssert($iErr = 1, ...)
Incorrect: _TestFmkAssert(@error = 1, ...) (already reset by the time _TestFmkAssert runs)

Related

_TestFmkHeader, _TestFmkRun, _TestFmkSummary

Example

#include <TestFramework.au3>

Func _TestMyFunc()
    _TestFmkHeader("Test: MyFunc")

    ; Basic assertion - always pass $vActual and $vExpected
    Local $iResult = 2 + 3
    _TestFmkAssert($iResult = 5, "2 + 3 equals 5", $iResult, "5")

    ; Asserting on @error - capture it immediately before any other call
    FileRead("nonexistent_file.txt")
    Local $iErr = @error
    _TestFmkAssert($iErr <> 0, "FileRead on missing file sets @error", $iErr, "<> 0")
EndFunc

_TestMyFunc()

Console output (both passing):

+ --- Test: MyFunc ---
> [PASS] 2 + 3 equals 5
> [PASS] FileRead on missing file sets @error

Console output (first assertion failing):

+ --- Test: MyFunc ---
! [FAIL] 2 + 3 equals 5 (expected: 5, actual: 99)
> [PASS] FileRead on missing file sets @error

Function Reference

_TestFmkHeader

Prints a labeled section header to the console to identify a group of related tests.

#include <TestFramework.au3>
_TestFmkHeader ( $sTitle [, $sColor = $TFW_COLOR_ORANGE] )

Parameters

$sTitleThe title text to display in the header.
$sColor[optional] Console color constant for the header line. Default is $TFW_COLOR_ORANGE.
Accepted values: $TFW_COLOR_ORANGE, $TFW_COLOR_YELLOW, $TFW_COLOR_BLUE, $TFW_COLOR_RED, $TFW_COLOR_WHITE.

Return Value

None.This function does not return a value.

Remarks

Call _TestFmkHeader() once at the beginning of each test function to clearly separate groups of related assertions in the console output. The header is printed with a blank line before it and surrounded by --- markers for readability.

The color constants control how SciTE colors the line in its output pane. See Constants Reference for details on the available colors.

Related

_TestFmkAssert, _TestFmkRun, _TestFmkSummary, $TFW_COLOR_*

Example

#include <TestFramework.au3>

Func _TestMyFunc()
    _TestFmkHeader("Test: MyFunc")
    _TestFmkAssert(True, "This assertion always passes")
EndFunc

_TestMyFunc()

Console output:

+ --- Test: MyFunc ---
> [PASS] This assertion always passes

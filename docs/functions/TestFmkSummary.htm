Function Reference

_TestFmkSummary

Prints the final Total/Passed/Failed summary block to the console.

#include <TestFramework.au3>
_TestFmkSummary ( )

Parameters

None.This function takes no parameters.

Return Value

None.This function does not return a value.

Remarks

Call _TestFmkSummary() once at the end of your _RunAllTests() function, after all test functions have been run via _TestFmkRun(). Calling it inside individual test functions would produce partial/incorrect totals.

The Failed line is printed in red ($TFW_COLOR_RED) when there are one or more failures, and in white ($TFW_COLOR_WHITE) when the count is zero, making a fully-passing suite immediately distinguishable from a suite with failures in SciTE's output pane.

The summary block is bounded by ============================== separator lines to visually separate it from the individual test output above.

Related

_TestFmkHeader, _TestFmkAssert, _TestFmkRun

Example

#include <TestFramework.au3>

Func _TestSomething()
    _TestFmkHeader("Test: Something")
    _TestFmkAssert(True,  "This passes",  "True",  "True")
    _TestFmkAssert(False, "This fails",   "False", "True")
    _TestFmkAssert(True,  "This passes",  "True",  "True")
EndFunc

Func _RunAllTests()
    Local $bAllPassed = True
    $bAllPassed = _TestFmkRun(_TestSomething, $bAllPassed)
    _TestFmkSummary()
EndFunc

_RunAllTests()

Console output:

+ --- Test: Something ---
> [PASS] This passes
! [FAIL] This fails (expected: True, actual: False)
> [PASS] This passes

==============================
+ Total: 3
> Passed: 2
! Failed: 1
==============================

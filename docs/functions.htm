Function Reference

_TestFmkHeader Prints a labeled section header to the console to identify a group of related tests.
_TestFmkAssert Evaluates a condition, records pass or fail, and prints the result to the console.
_TestFmkRun Runs a test function by reference and accumulates its pass/fail result into a cumulative boolean.
_TestFmkSummary Prints the final Total/Passed/Failed summary block to the console.

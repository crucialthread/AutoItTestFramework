TestFramework.au3

Click a function name for full documentation.

_TestFmkHeader Prints a labeled section header to the console to identify a group of related tests.
_TestFmkAssert Evaluates a condition, records pass or fail, and prints the result to the console.
_TestFmkRun Runs a test function by reference and accumulates its pass/fail result into a cumulative boolean.

Testable.au3

Provides _Tstbl_* wrapper functions for AutoIt built-ins. Include Testable.au3 in script code and use these wrappers instead of AutoIt built-ins directly so that calls can be intercepted by stubs in tests. See the Testable Reference for the full list of wrappers by category.

_Tstbl_MsgBox Testable wrapper for MsgBox.
_Tstbl_InputBox Testable wrapper for InputBox.
_Tstbl_FileExists Testable wrapper for FileExists.
_Tstbl_FileDelete Testable wrapper for FileDelete.
_Tstbl_RegRead Testable wrapper for RegRead.
_Tstbl_RegWrite Testable wrapper for RegWrite.
_Tstbl_GUICreate Testable wrapper for GUICreate.
_Tstbl_GUIGetMsg Testable wrapper for GUIGetMsg.
_Tstbl_ShellExecute Testable wrapper for ShellExecute.
_Tstbl_* Many more wrappers available — see Testable Reference for the complete list.

Each wrapper routes the call through a function pointer global ($g_hFn_*). In script code these point to the real AutoIt built-ins. In test files, including Stubs.au3 automatically rewires them to stubs.

Stubs.au3

Include Stubs.au3 in test files to automatically wire all _Tstbl_* wrappers to stubs. Stubs record every call with its parameters and return controlled values so tests can verify behavior without executing real AutoIt built-ins. See the Stubs Reference for full details.

_ResetStubs() Clears all recorded calls and configured return values. Call between tests to start with a clean state.
_SetStubReturn($sType, $iIdx, $vValue) Pre-configures the return value for the Nth call of a given stub type. Call before running the code under test.
$g_StubCalls["TypeName"] Map of recorded calls for a stub type. Each entry is keyed by 1-based call index and holds the parameters passed to that call. Also has a .count property.
$g_StubReturns["TypeName"] Map of pre-configured return values keyed by 1-based call index. Set via _SetStubReturn().
